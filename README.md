예전엔 책을 읽는 것을 좋아했다. 그런데, 코딩하는 직업을 가진 이후로는 책을 못읽는다. 책을 읽으려 노력하는데, 하루 날잡고 개발공부는 손 놓고 읽어야 한다. 그래서 리디북스 오디오 기능을 사용하게 됬는데, 일하면서도 책을 들을수 있어서 개꿀이긴 하다. 물론 놓치는 부분이 있는데, 놓쳤던 부분들을 확인해뒀다가 산책할 때 듣는다.<br>
